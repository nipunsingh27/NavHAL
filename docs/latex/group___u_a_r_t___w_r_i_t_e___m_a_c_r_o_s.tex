\doxysubsection{Detailed Description}
Simplified macros to write different types to UART using {\ttfamily \+\_\+\+Generic}. 

These macros provide a type-\/safe interface that automatically selects the appropriate write function based on the argument type. Supports\+:
\begin{DoxyItemize}
\item All integer types (char, short, int, long)
\item Floating point types (float, double)
\item String types (char \texorpdfstring{$\ast$}{*}, const char \texorpdfstring{$\ast$}{*})
\end{DoxyItemize}
