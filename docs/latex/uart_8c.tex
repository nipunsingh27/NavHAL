\doxysubsection{Detailed Description}
UART HAL implementation for Cortex-\/\+M4 (STM32\+F401\+RE) 

This file provides the implementation of UART communication functions for STM32\+F401\+RE microcontroller. It supports USART1, USART2, and USART6 peripherals with the following features\+:
\begin{DoxyItemize}
\item Baud rate configuration
\item Character, integer, float, and string transmission
\item Character reception
\item Data availability checking
\item Blocking (polling) implementations
\end{DoxyItemize}

The implementation uses GPIO alternate functions for UART pins and provides both generic (UART instance parameter) and specific functions.

\begin{DoxyNote}{Note}
All functions are blocking and will wait for hardware operations to complete 

Default configuration\+: 8 data bits, no parity, 1 stop bit
\end{DoxyNote}
