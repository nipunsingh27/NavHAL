\doxysubsection{Detailed Description}
UART initialization and I/O functions for STM32\+F4 UART1, UART2, and UART6. 

This file provides basic UART setup and read/write functionality for three UART peripherals. It supports initialization with baudrate, character, integer, float, and string transmission, as well as reading characters and checking availability.
