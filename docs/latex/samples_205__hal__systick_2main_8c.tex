\doxysubsection{Detailed Description}
Example\+: Multiple timers with callbacks and UART output. 

This example demonstrates\+:
\begin{DoxyItemize}
\item Configuring the system clock using HSI or PLL.
\item Initializing Sys\+Tick timer with 1 ms tick.
\item Initializing UART2 at 9600 baud for console output.
\item Setting up TIM2, TIM3, TIM4, TIM5, and TIM9 timers with interrupts.
\item Attaching callbacks to each timer to print messages via UART2.
\end{DoxyItemize}
