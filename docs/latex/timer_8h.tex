\doxysubsection{Detailed Description}
Timer and Sys\+Tick register defines and API for STM32\+F4 (Cortex-\/\+M4). 

This header contains register offsets, bit definitions, Sys\+Tick control, timer base addresses, and the public timer/\+Sys\+Tick API used by the HAL. All defines and function prototypes are intended for the STM32\+F4 series on a Cortex-\/\+M4 core.
