\doxysubsection{Detailed Description}
Example application\+: Generate PWM on PB10 using TIM2 and print duty cycle over UART2. 


\begin{DoxyItemize}
\item Initializes Sys\+Tick timer with 40 µs tick.
\item Initializes UART2 at 9600 baud.
\item Configures PB10 as alternate function (AF1) for TIM2\+\_\+\+CH3.
\item Initializes TIM2 for PWM output using HAL PWM driver.
\item Continuously ramps the duty cycle from 0\% to 100\% and prints it over UART2.
\end{DoxyItemize}
