\doxysubsection{Detailed Description}
Cortex-\/\+M4 NVIC (Nested Vector Interrupt Controller) register definitions. 

This header defines the memory-\/mapped structure for NVIC registers, including set-\/enable, clear-\/enable, set-\/pending, clear-\/pending, active bit, and priority registers. It also enumerates all Cortex-\/\+M4 processor exceptions and STM32\+F4-\/specific interrupt numbers for use in HAL and driver code.

\begin{DoxyNote}{Note}
The NVIC base address is 0x\+E000\+E100\+UL.
\end{DoxyNote}
