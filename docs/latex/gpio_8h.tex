\doxysubsection{Detailed Description}
HAL interface for GPIO control on STM32\+F4 series. 

This header declares high-\/level functions for configuring and using General-\/\+Purpose I/O (GPIO) pins, including mode setup, digital read/write, pull-\/up/pull-\/down configuration, RCC enabling, and alternate function selection.

These functions provide an abstraction layer over the low-\/level register access defined in {\ttfamily cortex\+\_\+m4\+\_\+gpio\+\_\+reg.\+h}.
