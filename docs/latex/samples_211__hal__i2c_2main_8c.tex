\doxysubsection{Detailed Description}
Example\+: Read raw temperature from BMP180 over I2C and print via UART2. 


\begin{DoxyItemize}
\item Initializes Sys\+Tick for delay functions.
\item Configures UART2 at 9600 baud for console output.
\item Configures I2\+C1 and GPIO pins for BMP180 communication (PB8=SCL, PB9=SDA).
\item Periodically starts a temperature measurement, reads the raw value, and prints it.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Delay between measurement and reading follows BMP180 datasheet (\texorpdfstring{$\sim$}{\string~}4.5 ms).
\end{DoxyNote}
