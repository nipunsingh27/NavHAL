\doxysubsection{Detailed Description}
Blink LED on PA05 and print system millis over UART2. 

This example demonstrates\+:
\begin{DoxyItemize}
\item Initializing Sys\+Tick timer with 40 µs tick.
\item Configuring GPIO PA05 as output for LED control.
\item Initializing UART2 at 9600 baud for console output.
\item Initializing TIM5 timer (prescaler 0, auto-\/reload 1e9).
\item Toggling LED and printing system millis in an infinite loop.
\end{DoxyItemize}
