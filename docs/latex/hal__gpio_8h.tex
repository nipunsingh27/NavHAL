\doxysubsection{Detailed Description}
Hardware Abstraction Layer (HAL) interface for General Purpose Input/\+Output (GPIO). 

This header file provides the interface for including the appropriate GPIO driver implementation based on the target architecture. For Cortex-\/\+M4 targets, it includes the {\ttfamily \doxylink{gpio_8h}{core/cortex-\/m4/gpio.\+h}} header.

The HAL GPIO module provides a platform-\/independent way to configure and control GPIO pins for input, output, and alternate functions.
