Detailed Description

Simplified macros to write different types to UART using _Generic.

These macros provide a type-safe interface that automatically selects the appropriate write function based on the argument type. Supports:

  • All integer types (char, short, int, long)
  • Floating point types (float, double)
  • String types (char *, const char *)

Example:

uart2_write(42); // Calls uart2_write_int
uart2_write(3.14f); // Calls uart2_write_float
uart2_write("hello"); // Calls uart2_write_string
