Detailed Description

Clock HAL implementation for Cortex-M4 (STM32F401RE).

This header provides register definitions, configuration structures, and function declarations to initialize and retrieve system and bus clocks on Cortex-M4 microcontrollers, specifically STM32F401RE in this implementation.

It supports configuring the system clock source including HSI, HSE, and PLL, PLL parameter setup, and clock prescaler controls for AHB and APB buses.

Note
This file is architecture-specific and included through the common hal_clock.h dispatcher based on target definitions (e.g., CORTEX_M4).
