Detailed Description

Register map and base addresses for GPIO ports on STM32F4 series.

This header defines the memory-mapped structure for GPIO port registers, as well as macros for accessing GPIO ports and pins on STM32F4 microcontrollers.

The GPIO ports are located at fixed base addresses in the AHB1 peripheral space. Each port consists of control registers for mode, type, speed, pull-up/pull-down, input/output data, bit set/reset, lock, and alternate functions.
